Data Coverage and Imputations

The SSPI uses the best available data to generate policy scores. When data is unavailable, we use imputations and data modelling to make scores as commensurable across countries and years as possible. The matrix below shows which indicator-year combinations have observed data (green) and which rely on imputed data (orange).

Imputations are typically carried out at the dataset level. Since each indicator relies on one or more datasets, the imputation percentage for an indicator is always greater than or equal to that for datasets. Toggle between an indicator and dataset views using the tab bar.
